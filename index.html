Programação para Todos

Do ZERO ao Básico

Objetivos e Público Alvo

Objetivo: Aprender a programar utilizando linguagem de programação javascript do ZERO ao básico que todo programador deve saber.


Público Alvo: Este projeto é direcionado a pessoas leigas em programação. Pessoas maiores de 18 anos, que tenham no mínimo o ensino médio incompleto, e o principal: queiram aprender a programar!


Neste curso você aprenderá a:

  • 1) elaborar soluções utilizando lógica de programação
  • 2) transformar problemas reais em programas de computador
  • 3) desenvolver programas usando o navegador web
